Eae, beleza?

IMPORTANTE: Nosso gerador online de Pessoa tem como intenção ajudar estudantes, programadores, analistas e testadores a gerar documentos necessários para testar seus softwares em desenvolvimento. No entanto, é crucial lembrar que a má utilização dos dados gerados é de total responsabilidade do usuário.

Os números e informações gerados são aleatórios e seguem as regras de criação de cada documento. Porém, é fundamental que você utilize esses dados de maneira ética e legal, respeitando a privacidade e os direitos das pessoas.

A extensão "Gerador de Pessoa", desenvolvida por Kléber Albino Moreira, utiliza APIs para gerar informações pessoais aleatórias, como nome e endereço. Ela faz uso da API do "Gerador Brasileiro" para obter dados aleatórios sobre pessoas brasileiras, é usado a API do "ViaCEP" para obter endereços reais com base na sigla do estado e no nome da cidade fornecidos pelo Gerador Brasileiro.

Além disso, Kléber modificou a lógica para gerar emails aleatórios, adicionando sua própria personalização ao processo.
